# Changelog

All notable changes to this project are documented in this file.

The project follows Semantic Versioning.

---

## v0.6.0 — Knowledge OS Interface

### Added

* Streamlit-based Knowledge OS interface.
* Interactive 3D Brain visualization.
* Searchable sidebar navigation.
* Markdown reading workspace.

### Changed

* Expanded the project from a Telegram-only workflow into a visual knowledge exploration interface.

---

## v0.5.0 — Local Storage

### Added

* Local Obsidian Vault integration.
* Safe filename generation.
* Telegram save confirmation.

---

## v0.4.0 — Markdown Generation

### Added

* Structured JSON validation.
* Markdown generation pipeline.
* Layered provider-agnostic architecture.

---

## v0.3.0 — Thought Refinement

### Added

* Structured thought refinement.
* Validation for non-thought messages.
* Title, Summary, Key Points, Tags, and Open Questions.

## v0.2.0 — LLM Integration

### Added

* Gemini/Gemma integration.
* Environment-based model configuration.

## v0.1.0 — Initial Release
