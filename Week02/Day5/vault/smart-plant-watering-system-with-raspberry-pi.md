# Smart Plant Watering System with Raspberry Pi
*Type: Research Idea*

## Summary
A project to develop an automated plant irrigation system that monitors soil moisture and provides remote notifications via Telegram, constrained by a specific budget.

## Key Points
- Hardware components: Raspberry Pi, soil moisture sensor, water pump, and tubing.
- Feature: Telegram integration for low-water reservoir alerts.
- Constraint: Total project budget is approximately $50.
- Requirement: Automated watering based on soil moisture levels.

## Open Questions
- What method or sensor should be used to measure the water level in the reservoir?
- How will the Raspberry Pi interface with the water pump (e.g., relay module)?
