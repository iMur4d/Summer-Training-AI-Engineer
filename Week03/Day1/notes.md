The article also mentions YAML (Yet Another Markup Language) Frontmatter. It stores metadata without putting it inside the document body.

Markdown is becoming part of AI workflows, not just documentation.

There are a lot of flavored Markdown: CommonMark spec (CMs) and GitHub Flavored Markdown (GFM)

Best Practices for Using Markdown: 1. Use a logical heading hierarchy, 2. Keep sections short and focused, 3. Specify the language in code blocks for syntax highlighting, 4. Don't overuse nested lists, 5. Use tables only for structured comparisons, 6. Keep formatting consistent.
